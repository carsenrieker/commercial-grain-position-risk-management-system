# Commercial Grain Position & Risk Management System

## Overview

This Excel-based project simulates the workflow of a commercial grain merchandising operation from contract origination through shipment and position reporting.

The workbook integrates contract management, scale ticket processing, quality discount automation, freight adjustments, position reconciliation, hedge monitoring, and dashboard reporting.

## Features

- Purchase Contract Management
- Sales Contract Management
- Inbound Scale Ticket Processing
- Outbound Scale Ticket Processing
- Quality Discount Automation
- Destination Quality Specifications
- Freight Adjustment Modeling
- Position Reconciliation
- Hedge Coverage Monitoring
- Dashboard Analytics

## Tools Used

- Microsoft Excel
- Lookup Functions
- Conditional Formatting
- Commodity Merchandising Concepts
- Risk Management Analysis
